# 🏥 HealthTrack – Mobile Health Tracking App

A mobile-first web app for tracking pain levels, managing medication reminders, booking doctor appointments, and accessing emergency contacts — all in one place.

## 📱 Features

- **Pain Level Logger** — Log your daily pain intensity (0–10 scale) with emoji feedback and notes. View your pain history at a glance.
- **Medication Reminders** — Set, snooze, dismiss, or mark reminders as taken. Supports medication, appointment, exercise, and hydration alerts.
- **Doctor Appointments** — Browse available doctors and book appointments directly from the app.
- **Pain Reports & Analytics** — View a visual chart of your pain trends over time with a detailed log history.
- **Emergency Access** — One-tap SOS button, call doctor, call emergency contacts, or start an emergency chat.
- **Caregiver Mode** — A dedicated view for caregivers to monitor a patient's pain level, average stats, therapy, and upcoming appointments.
- **User Profiles** — Patient and Caregiver registration with personalized dashboard.

## 🚀 Getting Started

### Prerequisites

No build tools or dependencies required. This is a pure HTML/CSS/JavaScript app.

### Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/HarshaSreeG/healthtrack-app.git
   cd healthtrack-app
   ```

2. Open `index.html` in your browser:
   ```bash
   open index.html
   ```
   Or just double-click the file in your file explorer.

### Deploy to GitHub Pages

1. Push to your GitHub repository.
2. Go to **Settings → Pages**.
3. Set source to `main` branch, `/ (root)`.
4. Your app will be live at `https://HarshaSreeG.github.io/healthtrack-app/`
